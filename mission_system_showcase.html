🎯 Dostępne Misje

Misje praktyczne: Strach przed stratą

  • 🧠 Dziennik emocji inwestora
    3-dniowe śledzenie emocji przed decyzjami inwestycyjnymi (25 XP)
  • 💰 Symulator małych inwestycji
    5-dniowa symulacja inwestycji z małymi kwotami (40 XP)
  • 📚 Analiza przypadków
    3-dniowa analiza przypadków strat inwestycyjnych (30 XP)

🚀 Przepływ Użytkownika

  1. Dashboard: Użytkownik widzi widget "Misje praktyczne: Strach przed stratą"
  2. Nawigacja: Kliknięcie "🎯 Idź do misji" przenosi do lekcji B1C1L1
  3. Auto-selekcja: Automatyczne wybranie zakładki "🎯 Misje praktyczne"
  4. Zarządzanie: Rozpoczynanie misji, przeglądanie zadań dziennych
  5. Postęp: Ukończenie zadań, zdobywanie XP, śledzenie postępu

🧪 Instrukcje Testowania

Aby przetestować system misji:

  1. Uruchom aplikację: streamlit run main.py
  2. Zaloguj się i przejdź do Dashboard
  3. Znajdź widget "Misje praktyczne: Strach przed stratą"
  4. Kliknij "🎯 Idź do misji"
  5. Sprawdź automatyczne przeniesienie do zakładki misji
